# Changelog

## [0.1.0] - Initial Release

### Added

* Built a local RAG pipeline using Ollama, ChromaDB, FastAPI, and Streamlit.
* Added PDF upload support through FastAPI.
* Added PDF text extraction using PyPDF.
* Added text chunking using LangChain RecursiveCharacterTextSplitter.
* Added local embeddings using Ollama `nomic-embed-text`.
* Added answer generation using Ollama `llama3.2`.
* Added ChromaDB vector storage and semantic search.
* Added Streamlit UI for uploading PDFs and asking questions.
* Added source previews with file name and page number.
