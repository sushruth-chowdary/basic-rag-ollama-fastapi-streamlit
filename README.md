# Basic RAG Project using Ollama, ChromaDB, FastAPI, and Streamlit

This is a local Retrieval-Augmented Generation project that allows users to upload PDF documents and ask questions from the uploaded content.

## Tech Stack

- Python
- Ollama
- Llama 3.2
- nomic-embed-text
- ChromaDB
- FastAPI
- Streamlit
- LangChain
- PyPDF

## Features

- Upload PDF documents
- Extract text from PDF pages
- Split text into chunks
- Generate embeddings using Ollama
- Store vectors in ChromaDB
- Ask questions from uploaded documents
- Display answers with source page previews
- FastAPI backend
- Streamlit frontend
